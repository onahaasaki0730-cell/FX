## ✨ 主要機能

### 1. **4つの時間足を同時表示**
- 15分足 📈
- 1時間足 📊
- 4時間足 📉
- 日足 📅

すべてのチャートが同時に更新され、トレンドを一目で確認できます。

### 2. **実際の市場データをリアルタイム表示**
- ローソク足チャート（日本式）
- 緑色 = 上昇（陽線）
- 赤色 = 下降（陰線）
- 60秒ごとに自動更新

### 3. **テクニカル指標を自動描画**

#### チャート上に表示される指標:
- **SMA(20)** - 青色の線：20期間単純移動平均
- **SMA(50)** - オレンジ色の線：50期間単純移動平均
- **ボリンジャーバンド** - 紫色の点線：価格変動の範囲

### 4. **サポート・レジスタンスレベル**
- **サポートライン** - 緑色の破線：価格を支える水準
- **レジスタンスライン** - 赤色の破線：価格の上昇を抑える水準

これらは自動的に計算され、チャート上に表示されます。

### 5. **エントリー/エグジットポイント（1時間足）**
- **エントリー価格** - 青色の実線：推奨購入価格
- **ストップロス** - 赤色の実線：損切りライン
- **テイクプロフィット** - 緑色の実線：利確目標価格

### 6. **トレンド表示**
各チャートの上部に現在のトレンドが表示されます:
- **↗ 上昇 XX%** - 買いトレンド（緑色）
- **↘ 下降 XX%** - 売りトレンド（赤色）
- **↔ レンジ XX%** - 方向性なし（オレンジ色）

### 7. **詳細分析パネル**（右サイドバー）

#### 💰 リアルタイム価格情報
- 現在価格
- 変動額・変動率
- 高値・安値

#### 💡 トレーディングシグナル
- 買い/売りの推奨
- 信頼度スコア（0-100%）
- シグナルの根拠
- 推奨エントリー価格
- ストップロス・利確目標

#### 📈 テクニカル指標の数値
- RSI（買われすぎ/売られすぎ）
- MACD（トレンド転換）
- 移動平均線の値
- ボリンジャーバンドの値

## 🎮 使い方

### ステップ1: ページを開く
```
https://8000-ipiyu6278hxw01fehjgee-5185f4aa.sandbox.novita.ai/charts
```

### ステップ2: 銘柄を入力
上部の入力欄に分析したい銘柄シンボルを入力:
- 株式: `AAPL`, `MSFT`, `GOOGL`, `TSLA`
- 為替: `USDJPY=X`, `EURUSD=X`
- 暗号通貨: `BTC-USD`, `ETH-USD`

### ステップ3: 「分析開始」をクリック
数秒で4つのチャートと詳細分析が表示されます！

### ステップ4: チャートを解釈

#### トレンドの確認
1. **4つの時間足を比較**
   - すべて上昇トレンド → 強い買いトレンド
   - すべて下降トレンド → 強い売りトレンド
   - バラバラ → 方向性が定まっていない

2. **移動平均線を見る**
   - 価格が移動平均線の上 → 上昇基調
   - 価格が移動平均線の下 → 下降基調
   - SMA(20)がSMA(50)を上抜け → ゴールデンクロス（買いシグナル）
   - SMA(20)がSMA(50)を下抜け → デッドクロス（売りシグナル）

3. **サポート・レジスタンスを活用**
   - 価格がサポートラインに接近 → 反発の可能性
   - 価格がレジスタンスラインに接近 → 下落の可能性
   - サポート/レジスタンスをブレイク → トレンド転換の可能性

4. **ボリンジャーバンドで判断**
   - 価格が上限に接触 → 買われすぎ（売り検討）
   - 価格が下限に接触 → 売られすぎ（買い検討）
   - バンドが狭い → ブレイクアウト前兆

5. **RSIで確認**（サイドバー）
   - RSI > 70 → 買われすぎ
   - RSI < 30 → 売られすぎ
   - 40-60 → 中立

6. **シグナルパネルで最終判断**
   - 買い/売りの推奨
   - 信頼度
   - 理由の説明
   - 具体的なエントリー価格

## 💡 実践例

### 例1: Appleを分析

```
1. https://8000-ipiyu6278hxw01fehjgee-5185f4aa.sandbox.novita.ai/charts を開く
2. シンボル入力欄に「AAPL」と入力
3. 「分析開始」をクリック

【チャートの見方】
- 15分足と1時間足が上昇トレンド
- 4時間足と日足はレンジ
→ 短期的には上昇だが、中長期は様子見

- 現在価格がSMA(20)の上にある
→ 短期トレンドは強い

- RSIが68（やや買われすぎ）
→ 一旦の利確や押し目待ちを検討

- サイドバーのシグナル: 「買い 60%」
- 根拠: MACD買いシグナル
→ 慎重に買いを検討
```

### 例2: ビットコインを分析

```
1. チャートページを開く
2. 「BTC-USD」と入力
3. 分析開始

【チャートの見方】
- すべての時間足が下降トレンド
→ 明確な売りトレンド

- 現在価格がボリンジャーバンド下限に接触
→ 売られすぎの可能性、反発の兆候

- レジスタンスラインが$98,000付近
→ 上昇してもこのレベルで抵抗

- サポートラインが$95,000付近
→ このレベルで下げ止まる可能性

→ サポートライン付近での反発を待つか、
   レジスタンス突破を確認してからエントリー
```

## 🎓 チャート分析のコツ

### 1. **上位足から確認**
日足 → 4時間足 → 1時間足 → 15分足の順で見る

### 2. **複数の指標を組み合わせる**
単一の指標ではなく、複数の視点で判断

### 3. **トレンドに逆らわない**
全体的なトレンドに沿った取引を心がける

### 4. **サポート・レジスタンスを重視**
重要な価格レベルでの反応に注目

### 5. **リスク管理を徹底**
必ずストップロスを設定
