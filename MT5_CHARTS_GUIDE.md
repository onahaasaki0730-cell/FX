# MT5スタイル FXチャート機能ガイド

## 🎯 概要

MT5（MetaTrader 5）スタイルのリアルタイムFXチャート機能を提供します。
主要な通貨ペアと金（ゴールド）の分析に特化し、複数の時間足を同時に表示できます。

## ⏱️ 選択可能な時間足

### **8つの時間足から最大4つまで選択可能:**

| 時間足 | 名称 | 用途 |
|--------|------|------|
| **1m** | 1分足 | スキャルピング、超短期トレード |
| **5m** | 5分足 | 短期デイトレード |
| **15m** | 15分足 | デイトレード、スイング初期分析 |
| **30m** | 30分足 | 中期デイトレード |
| **1h** | 1時間足 | スイングトレード（推奨） |
| **4h** | 4時間足 | スイング〜ポジショントレード |
| **1d** | 日足 | 長期トレンド分析 |
| **1w** | 週足 | 超長期トレンド把握 |

### **選択のベストプラクティス:**

#### **デイトレーダー向け:**
- 1m, 5m, 15m, 1h

#### **スイングトレーダー向け:**
- 15m, 1h, 4h, 1d

#### **ポジショントレーダー向け:**
- 1h, 4h, 1d, 1w

## 🚀 使い方

### **ステップ1: 通貨ペアを選択**

1. ページ上部の「通貨ペア」ドロップダウンから選択
2. デフォルトは「EUR/USD」

### **ステップ2: 時間足を選択（最大4つ）**

1. 8つの時間足から4つまでチェックボックスで選択
2. デフォルトでは「15m, 1h, 4h, 1d」が選択済み
3. 5つ以上選択しようとすると警告が表示されます

### **ステップ3: チャート表示**

1. 「チャート表示」ボタンをクリック
2. 4つのチャートが2x2グリッドで表示されます
3. 各チャートには以下が自動表示されます:
   - **ローソク足チャート**
   - **RSI指標**
   - **MACD指標**
   - **ボリンジャーバンド**

## 📈 画面構成

### **左側: チャートエリア（2x2グリッド）**

各チャートには以下が表示されます:

#### **ヘッダー:**
- 通貨ペア名と時間足
- トレンドバッジ（↗ 上昇 / ↘ 下降 / ↔ レンジ）
- トレンド強度（パーセンテージ）

#### **チャート本体:**
- **TradingViewウィジェット**使用
- MT5スタイルのダークテーマ
- リアルタイム価格更新
- ズーム・パン操作可能
- 日本時間（Asia/Tokyo）表示

#### **自動表示される指標:**
- RSI（Relative Strength Index）
- MACD（Moving Average Convergence Divergence）
- ボリンジャーバンド（Bollinger Bands）

### **右側: 分析パネル**

#### **1. 現在価格カード**
- リアルタイム価格
- 変動額と変動率
- 高値・安値

#### **2. マルチタイムフレーム分析**
- 選択した全時間足のトレンド方向と強度を一覧表示
- 各時間足の分析結果（上昇・下降・レンジ）
- トレンド強度（0-100%）

#### **3. トレーディングシグナル（1時間足ベース）**
- **シグナルタイプ:** 強い買い / 買い / 中立 / 売り / 強い売り
- **信頼度:** 0-100%のスコア
- **エントリー価格:** 推奨エントリーポイント
- **損切り（Stop Loss）:** リスク管理レベル
- **利確（Take Profit）:** 目標利益レベル
- **シグナルの根拠:** なぜこのシグナルが出たのか詳細説明

#### **4. テクニカル指標（1時間足ベース）**
- RSI: 買われすぎ/売られすぎ判定
- MACD: トレンド転換シグナル
- SMA(20): 短期移動平均線
- SMA(50): 中期移動平均線
- ボリンジャーバンド: 上限・下限レベル

## 💡 実践的な使い方のヒント

### **トレンド方向の確認:**
1. 全時間足のトレンドバッジを確認
2. 複数の時間足で同じ方向なら強いトレンド
3. 時間足ごとに方向が異なる場合は注意

### **エントリータイミング:**
1. 大きな時間足（4h, 1d）でトレンド方向を確認
2. 小さな時間足（15m, 1h）でエントリーポイントを探す
3. トレーディングシグナルの信頼度を確認
4. エントリー価格、損切り、利確レベルをチェック

### **リスク管理:**
1. 必ずストップロスを設定
2. 信頼度が高いシグナル（70%以上）を優先
3. 複数の時間足が同じ方向を示す時のみエントリー

### **ファンダメンタル分析との組み合わせ:**
1. 経済指標発表前後は注意
2. ニュースAPIで最新情報をチェック
3. センチメント分析を参考に

### **パフォーマンス:**
- チャート表示: 即座
- 分析データ取得: 1-2秒
- 自動更新: 継続的
